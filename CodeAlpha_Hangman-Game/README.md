# 🎮 CodeAlpha Hangman Game

A console-based Hangman game developed in Python as part of the CodeAlpha Python Programming Internship. The game features multiple difficulty levels, ASCII Hangman graphics, score tracking, and persistent high score storage with player names.

## 📌 Features

* Three Difficulty Levels

  * Easy
  * Medium
  * Difficult
* ASCII Hangman Visual Representation
* Score Tracking System
* Persistent High Score Storage
* High Score Holder Name Tracking
* Multi-Word Support
* Input Validation
* Replay Functionality
* Correct and Wrong Guess Tracking

## 🛠 Technologies Used

* Python 3
* File Handling
* Exception Handling
* Loops and Conditional Statements
* Lists and Strings
* Random Module

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/devkaranofficial/CodeAlpha-Hangman-Game.git
```

2. Navigate to the project directory:

```bash
cd CodeAlpha-Hangman-Game
```

3. Run the game:

```bash
python hangman.py
```

##  Gameplay

1. Start the game.
2. Choose a difficulty level.
3. Guess letters one at a time.
4. Avoid incorrect guesses to keep the Hangman alive.
5. Earn points by winning rounds.
6. Beat the existing high score and save your name on the leaderboard.

##  Scoring System

| Difficulty | Points Awarded |
| ---------- | -------------- |
| Easy       | +1             |
| Medium     | +2             |
| Difficult  | +3             |

# Email Address Extractor

## Project Overview

Email Address Extractor is a Python automation script developed as part of the CodeAlpha Python Programming Internship.

The application reads a text file, extracts all email addresses using Regular Expressions (Regex), and saves them into a separate output file.

This project demonstrates automation, file handling, and text processing using Python.

---

## Features

* Read data from a text file
* Extract email addresses using Regular Expressions
* Save extracted emails to a new file
* Display total emails found
* Simple and efficient automation script

---

## Technologies Used

* Python 3
* Regular Expressions (re)
* File Handling

# 🤖 Monday - Personal Assistant Chatbot

## 📌 Project Overview

Monday is a rule-based Personal Assistant Chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

The chatbot can interact with users, remember their name during the session, manage tasks, save notes, perform calculations, provide motivational quotes, tell jokes, display date and time, and maintain chat history.

This project demonstrates Python fundamentals, file handling, modular programming, and user interaction.

# Stock Portfolio Tracker

## 📌 Project Overview

Stock Portfolio Tracker is a Python-based console application developed as part of the CodeAlpha Python Programming Internship.

The application allows users to purchase stocks from a predefined list, calculate their investment value, maintain a portfolio summary, and save the final report to a text file.

---

## 🚀 Features

- Display available stocks and prices
- Buy stocks by entering stock name and quantity
- Case-insensitive stock search
- Calculate investment value automatically
- Maintain a portfolio of purchased stocks
- Display total investment value
- Save portfolio report to a text file
- Input validation and error handling

---

## 🛠️ Technologies Used

- Python 3
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Exception Handling
- File Handling
